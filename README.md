# kaggle-ROG2-Wellbore-Geology-Prediction

## 目的
水平坑井に沿った地質の予測モデルを作成する。  
水平坑井とは、一定距離垂直に地面を掘ったあとに徐々に角度をつけて目的とする地層方向に水平になるように掘り進めていく手法。

## 評価指標
RMSEで評価する。

## 変数
### {WELLNAME}__horizontal_well.csv 

|変数名|説明|
|---|---|
|WELLNAME|wellの名前|
|MD|測定深度（フィート）：地表から井孔までの全長|
|X|東経（フィート）：水平面上の空間座標|
|Y|北方向（フィート）：水平面上の空間座標|
|Z|真の垂直深度（フィート）：海面からの垂直距離|
|ANCC, ASTNU, ASTNL, EGFDU, EGFDL, BUDA|各種地層の予測深度（trainingのみ）|
|TVT|真の垂直厚さ（フィート）：目的変数。水平坑井の1フィートごとに手動で解釈された地質的位置|
|GR|ガンマ線（API）：岩石の自然放射能を測定するログ。|
|Input Target|入力ターゲット（フィート）：特徴量として提供された TVT のコピー。評価ゾーンについて NaN 値が含まれる。|

### {WELLNAME}__typewell.csv 

|変数名|説明|
|---|---|
|TVT|鉛直深度インデックス（フィート）：鉛垂直測の主要な深度基準。関連する水平坑井のTVT（地質的位置）に対応する。|
|GR|ガンマ線（API）：相関分析に使用される垂直方向のガンマ線シグネチャ。|
|Geology|地層ラベル：地質単位を示す分類ラベル（例：EGFDL、BUDA）。|
